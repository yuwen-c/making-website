## CSS Variable

1. set the namespace:
in css style part:
```
:root{
  --bound;
}
```
- root means that all the elements can use this variable.
- two dash -- means declare a variable.

2. use the variable
(take the 11. practice for example)
```
#fig{
width: var(--bound);
height: var(--bound);
}
```
