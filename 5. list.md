## list

-ul: UNordered list
  -li
  
-ol: Ordered list
  - li
  - change the list type:
  <ol type="a":
  1, A, a, I, i
  - start="35": start from whatever you want.
  
- <ul>, <ol> can be used in a <li> item. a nested structure.  
  
-dl: definition list
 
   <dl>

    <dt>tom</dt>
    <dd>nice guy</dd>

    <dt>john</dt>
    <dd>criminal</dd>

  </dl>
